SPRITE FILES NEEDED
===================
Drop high-resolution pixel art PNG images here.
The site will automatically use them instead of the canvas pixel art fallback.

Required files:
- jude.png       (Jude character, ~128x160px or larger, transparent background)
- bike.png       (Orange BMX bike icon, ~144x144px, transparent background)
- baseball.png   (Baseball icon, ~144x144px, transparent background)
- beyblade.png   (Beyblade/spinning top icon, ~144x144px, transparent background)
- dino.png       (Dinosaur icon, ~144x144px, transparent background)
- hotwheels.png  (Hot Wheels car icon, ~144x144px, transparent background)
- lego.png       (Lego bricks icon, ~144x144px, transparent background)
- gallery.png    (Camera/photo icon, ~144x144px, transparent background)
- tennis.png     (Tennis racket & ball icon, ~144x144px, transparent background)
- cricket.png    (Cricket bat & ball icon, ~144x144px, transparent background)

Style: High-res pixel art matching the Beyblade game win/loss screen aesthetic.
All images should have transparent backgrounds.
If a PNG is missing, the site falls back to the canvas pixel art version.
